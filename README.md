# Yet Another Memory Game


## Build with [Ionic](http://ionicframework.com/docs/)
```bash
$ sudo npm install -g ionic cordova
$ git clone
$ cd yamg
$ npm install 
$ ionic serve
```
Then build wanted platform

```bash
$ ionic cordova platform add android
$ ionic cordova run android
```
